# Gaps

- Bounded-norm + distributional critic recipe is shown for model-free SAC; whether the same stabilization carries over to *model-based* off-policy RL (DreamerV3-style imagination critics) where bootstrapping happens in latent rollouts is untested — could close the gap to GIRL-style world-model RL at FlashSAC's wall-clock budget.
- 2/1024 UTD ratio works under massively parallel sim (1024 envs); how does it degrade as parallelism drops to single-environment real-world data collection where each transition is irreplaceable? No data-efficiency curve at low env-count below the CPU-sim regime.
- Strong sim performance on Allegro / Shadow Hand dexterous manipulation, but **no real-robot manipulation transfer** — only humanoid locomotion is tested in sim-to-real; whether FlashSAC's stability properties survive contact-rich grasping under sim-to-real domain randomization is open.
- Asymmetric actor-critic with privileged critic state during sim training is used for sim-to-real; pure-proprioceptive critics (no privileged info) and vision-based sim-to-real are untested — the wall-clock advantage may shrink if the critic also has to learn from images.
- Adaptive reward scaling tracks running variance σ²_{t,G} and max G_{t,max}; behavior under non-stationary reward shaping (curriculum changes, weight rebalancing) is not characterized — could quietly destabilize the distributional critic.
- Noise Repetition Zeta-distribution exponent s is fixed but not ablated against task horizon / reward sparsity; whether the same s generalizes from 50-step humanoid stride episodes to 1000-step long-horizon manipulation is open.
- σ_tgt=0.15 unified entropy target works across embodiments tested, but all are continuous-action, mostly Gaussian-policy regimes; portability to mixture/discrete or hybrid action spaces (gripper bit + EE delta) is untested.
- Distillation potential not explored: whether the 2.5M-param critic's value information can be distilled into a smaller deployable actor with no critic at deployment is an open recipe (relevant for VLA-scale on-robot inference).
- Comparison to model-based wall-clock-strong baselines (DreamerV3 v3.x, TD-MPC2-Distill) only at fixed steps; pareto frontiers (asymptotic SR vs wall-clock) under matched compute on long-horizon sparse-reward tasks would clarify when off-policy *model-free* beats model-based.
- BatchNorm choice over LayerNorm is justified empirically (smoother loss landscape, lower critic-loss condition number) but the theoretical argument hinges on large-batch replay statistics — failure mode under small-batch fine-tuning (e.g., online deployment adaptation) is unaddressed.
- Single GPU (RTX 5090) measurement; multi-GPU scaling curve and per-GPU memory profile (10M-transition buffer + 2048 batch) not reported, making lab-scale reproducibility hard to estimate.
