# Gaps

- **Leakage-free ≠ action-centric.** The target encoder (frozen V-JEPA2) was pretrained on generic video without embodiment/action supervision; blocking future-frame input to the student still leaves the *target* pixel-biased. An action-aware target (e.g., contact masks, hand pose, flow against gripper) is untested.
- **No dual-branch posterior comparison.** VLA-JEPA only contrasts against frame-difference latent actions (LAPA, UniVLA, villa-X). It does not compare to hand-motion-conditioned posteriors (Being-H0/H0.7). Whether JEPA's leakage-free constraint is compatible with a privileged-hand-pose teacher branch is open.
- **Human-video regression unexplained.** On SimplerEnv Google Robot, VLA-JEPA *without* human videos scores 78.4% vs 65.2% with them. The paper acknowledges but does not diagnose this — suggests L_WM on SSv2 can actively misdirect latents when downstream embodiment is far from human hands. A gating / retrieval / curriculum fix is an open direction.
- **Video diversity untested on the stated failure mode.** Only SSv2 (short, centered, atomic) is used. The Introduction's failure mode #2 (camera motion / nuisance background in in-the-wild video) is never empirically probed — no Ego4D, EgoExo4D, or YouTube-scale experiment.
- **Horizon T is brittle.** Ablation shows T=8 best; T=4 and T=16 each hurt different task classes. No mechanism for adaptive horizon selection; no test of hierarchical / multi-scale latent-action tokens.
- **Frozen target is a bottleneck.** The world-state encoder F(·) is frozen throughout. EMA-updated targets or co-adapted targets (common in JEPA-v2, I-JEPA) are not explored; whether co-adaptation re-introduces representation collapse under the leakage-free constraint is open.
- **Textual reasoning gap.** Real-world failures attributed to "lack of fine-grained reasoning over textual instructions." VLA-JEPA has no CoT / sub-goal / object-grounding module; whether latent-action pretraining composes with VLM-side reasoning supervision is untested.
- **β is a single global scalar.** L = L_FM + β L_WM uses one hyperparameter; per-token or per-timestep weighting (e.g., higher L_WM on contact frames) is untested.
- **Single-embodiment real-world eval.** 100 demos on one Franka + Robotiq setup; no cross-embodiment (bimanual, mobile, humanoid) validation of whether the JEPA latent space actually transfers, or whether L_WM simply overfits the pretraining embodiment distribution.
- **No falsifier for "leakage."** The claim that LAPA/UniVLA latents collapse to target-image compression is supported only by attention visualizations. A quantitative probe (e.g., reconstruction of future frame from latent action alone) would strengthen or falsify the central motivation.
